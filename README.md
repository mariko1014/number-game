# 数当てゲーム 🎮

## 概要
1〜100の数字を当てるゲームです。

## 遊び方
- 数字を入力
- ヒントを頼りに当てる
- 入力された数値「より大きい」か「より小さいか」が表示される。
- 正解したら、何回で正解にたどり着いたかがわかる。
- 
- ー4/29追加機能ー
- 回数制限
- 設定した回数以内に正解にたどり着けば成功。
- 答えにたどり着けなかったら、正解の数値が表示される。
- maxAttemptsの数値を変えて、制限の調整可能
